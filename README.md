## Polymer Starter Kit

## Prerequisites

1. Node.js
2. npm
3. bower
4. gulp

## Intrucctions

- Clone de repo and in the root directory:
```sh
bower install
```

And

```sh
npm install
```

Finally to run the site:

```sh
gulp serve
```
